# 2c-tools

这是我的 AI 工具箱仓库，用来整理、沉淀和持续迭代我会使用与开发的各类 AI tools。

这里会逐步收录提升效率、内容创作、信息整理、数据处理与自动化协作的实用工具。

## 规划中的 Tools

| Tool 名称 | 概括功能 | 状态 |
| --- | --- | --- |
| Prompt Studio | 管理、测试和迭代常用 Prompt 模板，方便复用与优化。 | 规划中 |
| Research Copilot | 聚合资料、提炼重点并输出结构化调研结论。 | 规划中 |
| Content Repurpose | 将一份原始内容改写成摘要、标题、社媒文案等多种版本。 | 规划中 |
| Data Cleaner | 清洗结构化数据，并生成可直接使用的表格与摘要。 | 规划中 |
| Repo Reviewer | 快速分析代码仓库，输出结构梳理、风险提示和改进建议。 | 规划中 |
| Workflow Agent | 将重复任务编排成可复用的 AI 自动化工作流。 | 规划中 |

后续会持续补充更多 tools 和使用说明。
